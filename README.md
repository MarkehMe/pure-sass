# Pure Sass
This is an up to date version of Pure Sass from the Sassinate-Pure tool.

# Examples

## Import base and grids
```sass
@import 'pure/base';
@import 'pure/grids';
```
## Import base, menus, and grids
```sass
@import 'pure/base';
@import 'pure/menus';
@import 'pure/grids';
```
## Import everything
There is a simple `all.scss` file to simplify this for you
```sass
@import 'pure/all';
```
